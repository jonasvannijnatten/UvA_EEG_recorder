General information

This tool can be used to prepare plots of specific parts of a data set.

The top panel shows the file name and the file size (dimensions) of the loaded dataset. In the bottom panel you can change settings before plotting the (selected) data.

In the onset field you should fill in the number of seconds you cut before each event (based on the number of samples you defined in the Cutting tool).

 

Input

The input should be an EEG MATLAB structure, containing data from this domain:

       time EEG time series (EEG recording representing Voltage over time).

The data in the struct should be 2D and have the following dimensions (check this in the struct under dims):

       1st dimension is samples (time);

       2nd dimension is channels;

 

Plotting

To plot the data, you simply press Plot. With the default settings, it will plot the first 8 columns (in this case electrodes), use 0 as onset and use the default axes settings.

You can also select specific channels you would like to plot and define when the onset of the event should be (based on the number of samples you defined in the Cutting tool). It is also possible to choose a window of interest by setting the value of the X-axis and Y-axis.

 

Output

This tool does not have the option to save the data, since no manipulation can be carried out. The tool does give you a plot, which can be saved under the tab File -> Save or Save as.
