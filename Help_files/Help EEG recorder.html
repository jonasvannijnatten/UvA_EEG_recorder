General information

EEG recorder for MOBIlab+ contains the following main functions:

       Generate a live view of EEG (in time domain and frequency domain)

       Save EEG to RAM or hard disk

       Configure DIO

       Analyze EEG data using several toolboxes

 

 

Input

To record EEG no input is needed. However, it is possible to load previously recorded data to view it in the plots. In this case, the input should be an EEG MATLAB structure, containing data from this domain:

       time EEG time series (EEG recording representing Voltage over time).

The data in the struct should be 2D and have the following dimensions (check this in the struct under dims):

       1st dimension is samples (time);

       2nd dimension is channels.

EEG recorder

How to record EEG?

Follow these steps to record EEG:

1.     Indicate how many seconds you want to record.

2.     Press Start EEG.

3.     Wait (about 15 sec!)..until the EEG appears in the graphs.

4.     Let the participant perform the task.

5.     After the experiment finishes, press Stop EEG. The entire EEG is now plotted.

6.     Save the data for further analysis under the tab File -> Save.

 

 

Function of Stop EEG button

When you press the Stop EEG button, it

       stops recording,

       stops analogue input,

       keeps data in RAM,

       and keeps the figure as it was when the stop button was pressed.

 

Function of Clear button

When you press the Clear button, it

       Stops recording

       Stops analogue input

       Clears analogue input object from memory

       Clears all data from memory

       Clears figures

 

Adjusting live view plots

 

Preview sample length

The number of samples that are plotted in the time domain figure. The larger the number, the more data is plotted. Examples:

       Setting preview sample length to 256 samples (= 1 second) then constantly the latest second of data is plotted.

       A preview length of 1024 samples will plot the latest 4 seconds of recorded data.

 

FFT sample length

The number of samples that is used to generate the FFT (Fast Fourier Transformation) and plot the signal in the frequency domain.

       The FFT sample length should always be smaller or equal to the preview sample length. (FFT length <= preview length)

 

Channel spacer

This number determines the distance between the plotted channels.

       Smaller number = bigger signal amplitude

 

FFT frequency range

Set the range of frequencies that are plotted live in the Powerspectrum window (frequency domain).

Output

The output of the EEG recorder after recording or using one of the tools is a MATLAB structure that has the following elements:

EEG

  • filename name of file on computer
  • fsample sampling rate 
  • dims names of the dimensions, e.g., samples, channels, trials etc 
  • domain type of data, this can be either time, frequency or tf (time-frequency)
  • data the actual data 
  • channelLabels optional names for channels 
  • channelTypes type of channel, this can be either EEG, EMG, EOG, ECG or Marker 
  • sampleinfo the first and last sample number of the original dataset (this remains the same, even after cutting or other manipulations)
  • time timestamps per sample
  • history list of manipulations carried out on the data 
  • Optional: frequency frequencies (this element is added after TF analysis)
