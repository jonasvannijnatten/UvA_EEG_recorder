Time-Frequency Analysis

This function generates a time-frequency representation of an EEG time series with the choice of several baseline correction methods, and the option to calculate the average power within a given time-window and frequency range of interest.

Input

The input should be an EEG time series (EEG recording representing Voltage over time) consisting of one or more channels, trials or subjects.

The first dimension should represent time (samples).

The second dimension should represent the channels.

The third dimension should represent either trials or subjects.

 

Output

This function plots the time-frequency representation in the upper left axis.

Baseline correction methods

·        None
No baseline correction is applied and the function plots the power density (PSD) estimates. These PSD values have no unit.

·        Power  / baseline
This method calculates the power relative to the baseline period per frequency. First the average baseline power is calculated by averaging the power over the baseline period (see Baseline sample(s)) per frequency. Then the power of each time point is divided by the average baseline power (again per frequency).
The resulting values lie around 1, where values greater than 1 represent a relative increase in power, and values less than 1 represent a decrease in power.
Note: this method is most useful to show changes over time within a frequency band of interest.

·        Normalized power
This method calculates the normalized power by dividing the power of each frequency by the total power. This is done per time point.
The resulting values lie between 0 and 1 and represent the relative contribution of each frequency to the total power.
Note: This method is most useful to show which frequencies are most present at each time point.
