Help Data Plotter

 

 

General information

Use this function to prepare plots of specific parts of a data set.

Top panel shows:

·         The name of the loaded data set

·         Size of the loaded data set

·         Adjustable sample rate

 

Plotting:

·         Select if you want to plot rows or columns.

·         Select the rows/columns you want to plot (hold the mouse over this field for Tooltip)

·         Set X & Y axis: This is optional. When set, only the set part of the plot will be displayed. Automatic scaling is used to fit the data when left empty.

·         X-axis is in seconds (calculated from the sample rate)

·         Y-axis is the amplitude of the signal in microVolts (10^-6)

·         Set event onset time (in seconds). This depends on how many samples you cut before the event in the cutting tool.

·         Plot, plots the selected data in one axis.

 

 

Examples

Setting 1 (Default):

Plot columns

Onset = 0

X axis = empty

Y axis = empty

Columns = 1:8

 

Setting 2:

Plot Columns

Onset = .025s

X axis = [-0.25 0.8]              (time from -.25s - .8s)

Y axis = [-25 35]                   (amplitude from -25 – 35 microVolts)

Columns [3 5 8]                   (channels 3, 5 and 8)
