General information

The top panel shows the name and the size of the loaded data set.

This tool can be used to:

·         Block noise such as 50 Hz

·         Block all other frequencies except the frequency of interest

·         Measure the energy content of a frequency band function moved to spectral analysis

 

Input

Filters accepts data in the following formats:

·         A column vector

·         A 2D array (rows representing samples, columns representing channels/trials)

 

Filter settings

Select to which columns to apply the filter

·         First 8 à use this when filtering a dataset with 8 EEG channels plus digital channels (you don’t want the filter to affect the digital signal)

·         All à use when the columns represents single trials, which usually are more than 8

The filter has two settings

·         Band-pass filter

o   Blocks all frequencies expect the frequencies between “Filter Frequencies”

·         Band-stop filter

o   Allows all frequencies expect the frequencies between “Filter Frequencies”

 

The filter explained

There are many different filters. Filtering an incoming data stream is often achieved using a hardware filter such as the one installed in the EEG data acquisition set-up but it can also be achieved in software. Filtering already gathered data is usually achieved using software filters. It is possible to filter a signal in the time-, or in the frequency domain.

The filter you are using now filters in the frequency domain.

 

In short the filter takes the following steps:

1.    It takes the FFT of the signal

2.    It makes all frequency components except the desired components 0

3.    It performs the inverse FFT to generate the filtered signal.

 

The example below illustrates the procedure:

The signal y(t) consists of the sum of a 20 Hz sinusoid and a 50 Hz sinusoid and some random noise. We will block all frequencies between 45 and 50 Hz.

Using the fast Fourier transform (FFT) we obtain the single sided amplitude spectrum |Y(f)|. It clearly shows the two frequencies that were used to construct this signal.

The Fast Fourier transform of a signal consists however of coefficients that have a real and a complex part. For the plot above we only plotted the absolute real values and we omitted their symmetric second part and the complex part of |Y(f)|. So you just see the first half of real |Y(f)| . We also adjusted the frequency scale using our sample rate. In this way the peaks correspond to the observed frequencies.

Output

Filter generates three types of output

·         A vector or array of filtered input

·         A vector containing the energy

·         Optional plots of the original and filtered signal, both in the time and frequency domain.   

 

Using the output:

·         Filtered signals can be used for further spectral or ERP analysis.

·         Each element in the energy vector corresponds to the variance in the corresponding column of the data array. Variance is a measure for the energy in the signal. This function is mainly useful if one filters for example Alfa or the Theta.

·         Use plots to fine-tune filter settings.
