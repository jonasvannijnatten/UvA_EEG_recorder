General information

The top panel shows the name and the size of the loaded data set.

In the second panel you can select the channels you want to view in the tools below. Note that the trial browser allows you to view multiple channels at the same time, while the ERP assessment can only view one channel at a time.

In the “onset” field you should fill in the number of samples you cut before each event (the number of samples you defined in the Event Cutter tool).

 

 

Trial level analysis

This is used to inspect and correct data sets of individual subjects on the level of single trials.

For these functions you need to load 3D data of a single subject, and the dimensions should represent the following:

·         Row               = samples

·         Column         = channels

·         3rd                   = trials

It allows you to:

·         Go scroll all single trials of either the raw or baseline corrected data

·         Apply baseline correction

·         Apply automated trial rejection still unavailable

Select whether you want to inspect the uncorrected or corrected trials and click “browse trials”. A new window will pop up (the trial browser) that plots the cut out EEG signal per trial, for channels you selected. A vertical line indicates the moment of stimulus onset as defined by the “onset”.

Baseline correction takes the average signal amplitude of the samples before the onset and subtracts this from the complete signal. This is done separately for each trial and each channel. If you check the box “show before and after plots” for each channel it will plot two figures, for the uncorrected and corrected data, showing all trials single trials in blue and the average of all trials in red. If baseline correction is completed a green text will appear underneath the button indicating that the tool has the corrected data in memory and can be inspected using the trial browser and saved using the “save” function in the taskbar if you are happy with the result.

 

 

 

Subject level analysis

This is used to assess the measures of ERPs when they are averaged per subject.

For this function you need to load 3D data of a single subject, and the dimensions should represent the following:

·         Row               = samples

·         Column         = channels

·         3rd                   = subjects

Note: you can only select one channel at the same time to inspect (so only inspect 2D data).

It allows you to:

·         Scroll through the ERPs of different subjects

·         Select a time window around your ERP component of interest

·         Get information about:

-       The peak amplitude (for positive and negative peaks)

-       The peak timing (for positive and negative peaks)

-       The average amplitude

-       The under the curve

Fill in a lower and upper limit in milliseconds of your time window of interest. Click “Go!” and two red lines are drawn indicating indicating the lower and upper limits, two red stars indicate the location of the min and max values within the limits, and the bar on the right side displays the measures of the selected component. Write down the values you are using for your analysis and repeat this for all subjects.
