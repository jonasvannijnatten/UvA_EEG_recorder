General information

Top panel indicates the name and the dimensions of the loaded data file

The size of a data set is indicated by three numbers [1 2 3]

1.    The number of rows

2.    The number of columns

3.    The “depth” in the third dimension (This is 1 in a 2D data-set)

 The different subpanels allow you to change the shape and size of your dataset by making cuts, selections, averages, rotations and adding datasets together.

 

Datasets and dimensions

When recording an EEG signal the computer takes many samples of the incoming signal and writes a new value to a new row in the dataset for every sample moment. This happens simultaneous for all the channels which are written in different columns resulting in a 2D spreadsheet or table. After cutting the data into all the repetitions it puts the separate events behind each other in the 3rd dimension. So the raw EEG signal is a 2D dataset of which the rows represent the number of samples recorded, and the columns the number of channels. After cutting the data the 3rd dimension represents the number of TTLs/events. The size of each of these dimensions is shown in the top panel.

 

 

Transpose & Permute

Allows you to rotate the data matrix along three dimensions.

 

 

Resize

Allows you to ‘cut’ sections of interest from a data set.

Indicate for all three dimensions from where till where you want to keep your data.

In de example above:

            Row 1 till 512 are kept

            Column 1 till 7 are kept

            The third dimension “depth” 1 till 5 is kept

So, the final dimensionality of the cutout data section will be 512 - 7 - 5

 

Average

Allows you to average the dataset along the selected dimension after which that dimensions size will be 1.

 

Select

Allows you to make a selection of rows/columns/3rd that either you want to keep or discard.

In contrast to the resize tool this also allows you to select rows/columns/3rds that are not consecutive, e.g. select only channel 3, 5 and 7 instead of channel 3-7.

 

Concatenate

Allows you to add data sets together.

You can “glue” one data sat to another along one of three dimensions. Note that the dimension sizes, other than the one along which you concatenate, should be the same for both datasets. Say you want to put together two datasets in the 3rd dimension, then the nr of rows and columns should be equal. This means that you cannot “clue” a data set with 8 columns to one that has only seven columns. This rule applies to all dimensions.
