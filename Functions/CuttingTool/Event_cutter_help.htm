General information

Top section shows loaded file name, file size before the cut, file size after the cut.

The program can make repetitive cuts based on:

1.       TTL pulses

2.       User defined time

 

TTL based cuts

To cut out the EEG around an event:

1.    Select the DIO column (the channel number on which the TTL comes in)

2.    Select the number of samples you wish to cut before and after the event

e.g. if you want 1 second of data before the event of interest and 2 seconds after fill in 256 and 512 respectively

3.    Click the “Cut” button and save the data

 

Output:

Each time a TTL pulse is detected in the selected channel it will cut out the indicated time window around the event.

Out of the complete 2D data set it cuts out the first event (which is still 2D) after which all the repeating events are cut out and appended in the 3rd dimension. The number of cuts is equal to the number of TTLs detected.

After the cut the data is removed from memory.


Time based cuts

To cut out the EEG around a time event:

1.    Indicate the time (relative to the beginning of the recording) when the first event of interest takes place in the first box.

2.    Indicate the amount of seconds before and after the event you wish to cut from the EEG in the second and the third text box

3.    Indicate the period between the events of interest in the fourth box

4.    Indicate the number of cuts in the fifth box.

5.    Click the “Cut” button and save the data

 

Output:

Out of the complete 2D data set it cuts out the first event (which is still 2D) after which all the repeating events are cut out and appended in the 3rd dimension. The number of cuts is equal to the number of TTLs detected.

After the cut the data is removed from memory.
